# Пульт управления книгами

Единая программа для склада: выставлять и снимать лоты книг разом на Ozon, Wildberries и Avito.
Книги — б/у, единичные экземпляры (остаток = 1). Продажа на одной площадке автоматически снимает
книгу с остальных.

## Что уже готово (Фазы 0–1)
- Вход по паролю, общий каталог для нескольких ПК.
- Каталог книг: поиск, фильтры, карточка, массовые операции (выставить / снять / отметить проданной).
- Импорт выгрузок с площадок (CSV/XLSX) с сопоставлением колонок и дедупликацией по SKU/ISBN.

## Запуск локально (Windows)
```powershell
# 1. Один раз — окружение и зависимости
py -m venv .venv
.\.venv\Scripts\python.exe -m pip install -r requirements.txt

# 2. Настройки
copy .env.example .env
#   открой .env и поменяй APP_PASSWORD и SECRET_KEY

# 3. Запуск
.\.venv\Scripts\python.exe run.py
```
Открой http://127.0.0.1:8000 — попросит пароль (из APP_PASSWORD).

## Архитектура
- Backend: FastAPI, база через SQLAlchemy (SQLite локально → PostgreSQL на VPS, меняется только `DATABASE_URL`).
- Интерфейс: серверные страницы Jinja2, лёгкие для слабых ПК.
- Секреты площадок шифруются (Fernet), см. `app/security.py`.
