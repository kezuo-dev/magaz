# Пульт управления книгами

Единая программа для склада: отслеживает лоты книг на Ozon и Wildberries. Книги — б/у, единичные
экземпляры (остаток = 1). Продажа на одной площадке автоматически снимает книгу с другой —
продавцу ничего делать не нужно.

## Что делает
- Вход по паролю, общий каталог для нескольких ПК.
- Каталог книг для отслеживания: поиск, фильтры, карточка (только просмотр).
- Опрос заказов площадок каждую минуту → мгновенное авто-снятие проданной книги с другой площадки.
- Фоновая сверка всего каталога с Ozon/WB + кнопка «Обновить каталог» для ручной сверки.
- Импорт выгрузок с площадок (CSV/XLSX) с сопоставлением колонок и дедупликацией по SKU/ISBN.

## Запуск локально (Windows)
```powershell
# 1. Один раз — окружение и зависимости
py -m venv .venv
.\.venv\Scripts\python.exe -m pip install -r requirements.txt

# 2. Настройки
copy .env.example .env
#   открой .env и поменяй APP_PASSWORD и SECRET_KEY

# 3. Запуск
.\.venv\Scripts\python.exe run.py
```
Открой http://127.0.0.1:8000 — попросит пароль (из APP_PASSWORD).

## Что дальше
- Устойчивость на объёме 50k (лимиты запросов, повторы, бэкапы).
- Инкрементальная сверка каталога (по курсору updatedAt) вместо полной, чтобы чаще и дешевле.

## Архитектура
- Backend: FastAPI, база через SQLAlchemy (SQLite локально → PostgreSQL на VPS, меняется только `DATABASE_URL`).
- Интерфейс: серверные страницы Jinja2, лёгкие для слабых ПК.
- Секреты площадок шифруются (Fernet), см. `app/security.py`.
